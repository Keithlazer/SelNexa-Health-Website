# SelNexa Health Analytics Setup Checklist

Before paid traffic or outbound campaigns, confirm:

- [ ] Google Analytics 4 or Plausible installed site-wide.
- [ ] Google Search Console verified for `https://www.selnexahealth.com`.
- [ ] `sitemap.xml` submitted in Search Console.
- [ ] Microsoft Clarity installed if session replay is approved by privacy policy.
- [ ] Goals configured for demo bookings, pitch deck downloads, contact submissions, and pilot access requests.
- [ ] UTM convention documented for LinkedIn, email, partner, and investor outreach links.
- [ ] Privacy policy updated to name analytics providers and opt-out method.
